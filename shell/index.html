常用 Shell 命令

发表于 2018-12-25

zsh、bash 都是 shell，zsh 比 bash 更加强大。

ssh 连接远程服务器。例如： ssh xxx@192.168.*.*。

ssh Secure Shell（安全外壳协议，简称SSH）是一种加密的网络传输协议，可在不安全的网络中为网络服务提供安全的传输环境。 SSH通过在网络中建立安全隧道来实现SSH客户端与服务器之间的连接。

ls 是list的缩写，用来显示目标列表。
ll 是ls -lh的别名，列出的信息更加详细。

screen 使得可以同时连接多个本地或远程的命令行会话，并在其间自由切换。

cd .. 返回上一层目录。

pwd 命令以绝对路径的方式显示用户当前工作目录。命令将当前目录的全路径名称（从根目录）写入标准输出。

mv 移动命令。mv {文件/文件夹} {目标地址}。

rm 移除文件。移除文件夹时需要 rm -rf。

unzip 解压缩。

mkdir 创建文件夹。

--help 显示帮助信息；
--version 显示版本信息。
